# GlucoPlot

GlucoPlot is a medical monitoring system comprising a patient mobile app, a doctor web portal, and a secure backend.

## Project Structure

This is a mono-repository containing the following components:

- **[apps/web](./apps/web/README.md)**: React application for the Doctor Portal.
- **[apps/mobile](./apps/mobile/README.md)**: Flutter application for the Patient Mobile App.
- **[supabase](./supabase/README.md)**: Backend logic, database migrations, and Edge Functions.
- **[docs](./docs/README.md)**: Comprehensive project documentation.
